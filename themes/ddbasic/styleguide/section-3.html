3.1.1 #permalink Font

Mixin for setting typography

font($name)

Parameters:
  • $name
    Name of typography setting - is a string, possible string-values are:
  • 'base'
  • 'base-bold'
  • 'header-large'
  • 'header'
  • 'header-small'

3.1.2 #permalink Place-icon

Mixin for placing icon on element

place-icon($icon, $color, $size)

Parameters:
  • $icon
    Name of icon to be placed - is a string, see _mixin.scss for icon names
  • $color
    Color of icon - default is $charcoal-opacity-dark
  • $size
    Size of icon - default is 54px

3.1.3 #permalink Button

Mixin for making button with icon

button($icon, $color, $background-color, $color-hover, $background-color-hover)

Parameters:
  • $icon
    Name of icon to be placed - is a string, see _mixin.scss for icon names
  • $color
    Color of text and icon - default is $color-text-on-secondary
  • $background-color
    Background-color of button - default is $color-secondary
  • $color-hover
    Color of text and icon on hover - default is $white
  • $background-color-hover
    Background-color of button on hover - default is $grey-dark
Source: configuration/_mixin.scss, line 116
