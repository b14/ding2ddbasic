2.1.4 #permalink Submit-button with icon

In order to place icon on submit-button we wrap it in a div with the class "submit-button-with-icon" and place an empty div before it with the class "color-and-icon". Specific icons and hover styles are placed on specific classes in scss

Examples
Default styling
.example-icon
example icon
.example-icon:hover
example-icon hover 
<div class="submit-button-with-icon [modifier class]">
<div class="color-and-icon"></div>
<input type="submit" value="Submit button with icon">
</div>

2.1.5 #permalink Image styling 16:9

The class image-styling-16-9 keeps a 16:9 format for images in teasers even if the image is another format. This is done by adding a box around the image. The class can also be used on an element with a background-image

Example 
<div class="node-teaser">
<div class="image-styling-16-9">
<img src="" />
</div>
</div>
