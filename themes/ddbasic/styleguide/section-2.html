2.1.4 #permalink Submit-button with icon

In order to place icon on submit-button we wrap it in a div with the class "submit-button-with-icon" and place an empty div before it with the class "color-and-icon"

Example 
<div class="submit-button-with-icon">
<div class="color-and-icon"></div>
<input type="submit" value="Submit button with icon">
</div>
